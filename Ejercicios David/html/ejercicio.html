Párrafo con texto de ejemplo para probar diferentes etiquetas, como negrita, cursiva o subrayado. Podemos insertar enlaces a otras páginas web, como Google o YouTube.

Podemos también agrupar elementos dentro de la línea para, por ejemplo, cambiar el color de determinados elementos dentro del mismo párrafo. El párrafo también puede contener listas ordenadas.

  1. Elemento 1
  2. Elemento 2
Listas sin ordenar:
  • Elemento 1
  • Elemento 2
O diccionarios:
Término 1
Descripción.
Término 2
Descripción
